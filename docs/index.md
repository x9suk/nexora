## Quick Example

```nexora
// Hello World in Nexora
func greet(name) {
    return "Hello, ${name}!"
}

print greet("Nexora")
```

## Why Nexora?

- **Simple syntax** — Clean, readable code with minimal boilerplate
- **AI-native** — Built-in AI features for code generation and explanation
- **Fast** — Compiled and executed via a Rust-powered runtime
- **Expressive** — Pattern matching, string interpolation, closures, classes
- **Batteries included** — Standard library with math, strings, collections, HTTP, and more
