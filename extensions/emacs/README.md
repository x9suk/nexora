# Nexora for Emacs

Nexora language support for Emacs.

## Features

- Syntax highlighting for `.nx` files
- Font-lock keywords

## Installation

1. Clone this repository
2. Add to your `init.el`:

```elisp
(add-to-list 'load-path "/path/to/nexora-emacs")
(require 'nexora-mode)
```

3. Open any `.nx` file
