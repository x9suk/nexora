# Nexora for JetBrains

Nexora language support for IntelliJ IDEA and other JetBrains IDEs.

## Features

- Syntax highlighting for `.nx` files
- Code completion
- Structure view
- Go to definition
- Find usages

## Installation

1. Open IntelliJ IDEA
2. Go to Settings > Plugins
3. Search for "Nexora Language"
4. Click Install
5. Restart IDE

## Building from Source

```bash
./gradlew buildPlugin
```

The plugin will be in `build/distributions/`.
