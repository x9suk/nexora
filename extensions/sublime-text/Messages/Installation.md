# Nexora Syntax Highlighting - Installation

## Quick Install

1. Open Sublime Text
2. Go to **Preferences** > **Browse Packages...**
3. Create a new folder called `Nexora`
4. Copy `Nexora.sublime-syntax` and `Nexora.tmLanguage` into the `Nexora` folder
5. Restart Sublime Text

## Via Package Control

1. Open Command Palette (Ctrl+Shift+P)
2. Type **Package Control: Install Package**
3. Search for **Nexora** and install

## Verify Installation

1. Open any `.nx` file
2. Check that syntax highlighting is applied
3. If not, click the syntax selector in the bottom-right and select **Nexora**

## Notes

- The `.sublime-syntax` file is used by Sublime Text 3+
- The `.tmLanguage` file provides compatibility with older versions
- Both files provide the same highlighting rules
