# Nexora for Sublime Text

Nexora language support for Sublime Text.

## Features

- Syntax highlighting for `.nx` files

## Installation

1. Open Sublime Text
2. Go to Preferences > Browse Packages
3. Create a new folder `nexora`
4. Copy `Nexora.sublime-syntax` and `Nexora.tmLanguage` to the folder
5. Restart Sublime Text
