# OpenNexora Foundation

The OpenNexora Foundation is an independent non-profit organization dedicated to supporting the growth and development of the Nexora programming language ecosystem. Similar to the OpenJS Foundation for JavaScript, we provide a neutral home for critical infrastructure projects, ensure long-term sustainability, and foster collaboration across the community.

## Our Mission

**"To support the growth of the Nexora programming language ecosystem"**

We achieve this by:
- Providing governance and support for critical Nexora projects
- Ensuring neutrality and open collaboration
- Offering infrastructure and resources for project development
- Promoting diversity and inclusion in the community
- Maintaining high standards for code quality and security

## Foundation Projects

The OpenNexora Foundation currently hosts the following projects:

| Project | Description |
|---------|-------------|
| [Nexora Runtime](https://github.com/opennexora/nexora-runtime) | The core runtime environment for Nexora |
| [nxm Package Manager](https://github.com/opennexora/nxm) | The official package manager for Nexora |
| [nexora.dev Website](https://github.com/opennexora/nexora.dev) | Official documentation and community resources |
| [VS Code Extension](https://github.com/opennexora/vscode-nexora) | Official VS Code support for Nexora |

## Governance

The OpenNexora Foundation is governed by a Board of Directors and a Technical Steering Committee (TSC). For detailed governance information, see [GOVERNANCE.md](GOVERNANCE.md).

## Getting Involved

We welcome contributions from everyone! Whether you're a developer, designer, writer, or just interested in Nexora, there are many ways to get involved:

- **Contribute Code**: See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines
- **Join Working Groups**: Participate in specialized groups focused on specific areas
- **Attend Events**: Join our community meetings and conferences
- **Spread the Word**: Help us grow the Nexora community
