## Write for Us

We welcome guest contributions! If you'd like to write for our blog, please review our [Contributing Guidelines](../CONTRIBUTING.md) and submit a proposal to blog@opennexora.org.

### Writing Guidelines

- **Original Content**: All posts must be original and not published elsewhere
- **Technical Accuracy**: Ensure all code examples and technical details are accurate
- **Clear Writing**: Write in clear, concise English
- **Community Focus**: Focus on topics that benefit the Nexora community

### Post Format

We accept posts in Markdown format. Please include:
- A compelling title
- Author name and bio
- Publication date
- Category tags
- Featured image (optional)
